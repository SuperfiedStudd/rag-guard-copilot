# AGENTS

## Mission

Ship a clean, runnable v1 that demonstrates secure, identity-aware RAG for enterprise AI assistant scenarios.

## Primary agent responsibilities

- keep the demo runnable locally with minimal setup
- prefer clarity over abstraction
- make security decisions visible in the UI
- preserve mock data and deterministic behavior for interviews and demos

## Working norms

- default to local dependencies only
- keep optional model integrations disabled unless explicitly enabled later
- treat blocked retrieval and flagged injection as first-class outputs
- log meaningful events so reviewers can inspect system behavior quickly
