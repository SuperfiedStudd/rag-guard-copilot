# PROJECT CONTEXT

## Project

RAG Guard Copilot

## Purpose

Demonstrate how an enterprise assistant can perform retrieval while enforcing user-aware access policy, filtering malicious retrieved content, and masking sensitive data before prompt construction.

## Audience

- hiring managers
- security engineers
- AI platform teams
- technical interviewers

## Success criteria for v1

- app starts locally with Streamlit
- sample users and sample documents are bundled in-repo
- at least one query clearly shows allowed retrieval, blocked retrieval, injection detection, and PII masking
- audit activity is visible without external infrastructure
